# 추상 자료형

추상 자료형은 컴퓨터 과학에서 자료와 자료에 대한 연산을 명기한 것이다. 추상 자료형은 구현 방법을 명시하고 있지 않다. 이는 인터페이스와 구현을 분리하여 추상화 계층을 둔 것이다.
